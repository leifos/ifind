Experiment Instructions

Thank you for agreeing to participate in this experiment about online searching.

During this experiment we would like you to perform two different search tasks using the same search system.

Your Role

  • For this experiment imagine that you are a newspaper reporter, who needs to write a detailed report on the given topic.

The Search System

  • The search system that you will use connects to a database of about 1 million newspaper articles from the New York Times, Associated Press and Xinhua newspapers from 1996 to 2000.
  • The search system is similar to a web search engine; though there might be some differences to the search interface.

The Search Tasks

  • For each search task you will be given a short description of a topic and what information would be relevant to this topic.
  • When you view a particular newspaper article, please mark whether you think the document is relevant to the topic.
  • There are over 100 relevant documents in the database for each topic.

Before you start searching, we would like to ask you a few questions about your demographics.

Before each search task, we will ask you a few questions about your knowledge of the search topic, while after the task we will ask you about your search performance.

After you have performed each of the search tasks, we will ask you to complete a series of very short questionnaires about your search experience.

If you have any questions, please ask the experimenter now.
